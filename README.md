\# bilibter

一个实验项目，研究挖矿，不是大项目。

一个极简的个人货币系统实验项目。
\- 基于 SHA-256 工作量证明

\- 命令行交互，支持挖矿、转账、余额查询

\- 代码少于 200 行，适合理解区块链基础原理

\- main.py是未编译的版本

\- bilibter.exe是编译后的版本
\## 快速开始
```bash
python bilibter.py
然后输入 help 查看命令。
